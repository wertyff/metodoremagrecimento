# Orion Play Mobile

Projeto mobile real em Expo/React Native para Android e iPhone.

## Onde esta

- [mobile-app](</C:/Users/Administrator/Documents/New project/APP ORION PLAY/mobile-app>)

## Rodar localmente

```bash
npm install
npm start
```

## Abrir no celular

- Android: `npm run android`
- iPhone: `npm run ios`

## Gerar instalador

- Android APK de preview:

```bash
npm run build:android
```

- iOS build de preview:

```bash
npm run build:ios
```

## Observacoes

- O build de iPhone (`ipa`) normalmente usa nuvem/servico Apple.
- Em Windows, o fluxo de iOS fica pronto, mas o instalador final costuma depender de build remoto.
- O app foi estruturado com catalogo demo/editavel para futuras integracoes licenciadas e oficiais.
